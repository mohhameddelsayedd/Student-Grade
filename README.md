# 🎓 Student Grade Management System (C# Console App)

A simple and interactive console-based application built using **C#** to manage student grades.
This project demonstrates the use of fundamental programming concepts such as loops, conditions, collections, and basic CRUD operations.

---

## 🚀 Features

* 📥 Input multiple students (Name & Grade)
* 📊 Automatic grade classification:

  * ≥ 90 → Excellent
  * ≥ 75 → Very Good
  * ≥ 60 → Good
  * ≥ 50 → Pass
  * < 50 → Fail
* 📋 Display all students using `foreach`
* 📈 Calculate average grade using `while`
* 🏆 Find highest grade
* ❌ Show failed students
* ➕ Add new student
* ✏️ Edit existing student
* 🗑️ Delete student
* 📌 Menu-driven system using `do-while` and `switch`

---

## 🛠️ Technologies Used

* C#
* .NET Console Application
* Collections (`List<T>`)
* Control Structures (`for`, `foreach`, `while`, `do-while`, `if`, `switch`)
